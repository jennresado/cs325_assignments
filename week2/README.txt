To run stoogesort.py with "data.txt" file
1. $ python3 stoogesort.py
2. To run stoogesort.py, python3 is required
3. "data.txt" file structure follows
   a. first value of each line is the number of integers that need to be
      sorted, followed by the integers.
   b. Example values for "data.txt":
      4 19 2 5 11
      8 1 2 3 4 5 6 1 2
4. The output will be written to files called "stooge.out"
