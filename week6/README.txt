To run westlers.py with a text input file
1. $ python3 wrestlers.py <file_name>
   a. Replace <file_name> with name of text file with .txt file extension
2. To run wrestlers.py, python3 is required
3. text file structure follows
   a. The file contains the  number of wrestlers, n, followed by their 
      names, the number of rivalries r and rivalries listed in pairs. 

   b. Example values for text input file:
      6 
      Bear 
      Maxxx 
      Killer 
      Knight 
      Duke 
      Samson 
      5 
      Bear Samson 
      Killer Bear 
      Samson Duke 
      Killer Duke 
      Maxxx Knight 

4. Results are outputted to the terminal.
